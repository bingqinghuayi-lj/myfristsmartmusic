1：源码介绍：	
    inage 保存照片资源
    lyrics 保存歌曲的歌词信息
    song 保存歌曲的播放文件

2：该版本实现的功能：
    暂停，播放，静音，非静音，音量进度条控制歌曲音量，快进快退，单击按钮切换背景图片，歌词单行显示，歌词全屏并且可以实现滚动，歌词能和时间对应，进度条拖拽可以实现歌词时间的改变，
    双击歌单可以切换歌曲，顺序播放，可以显示歌曲总时间，当前播放时间，歌曲名，显示当前系统时间，可以添加收藏喜欢。

3：该版本的优点：
    功能基本实现。

4：待实现的地方：
    实现歌曲的随机播放，从网上下载歌曲。
